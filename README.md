# WtPTableGames

Экспериментальный проект с Телеграм-ботом для настольных игр. Репозиторий больше про эксперименты, чем про готовый продукт: здесь пробовал идеи, логику и взаимодействие с API.

## Идея

- Набросок бота, связаного с настольными играми
- Тестирование различных подходов к логике бота
- Площадка для экспериментов с Telegram Bot API на Kotlin

## Технологии

- Kotlin
- Telegram Bot API

## Как запустить

Проект не доведён до production-состояния.  
Основной вариант — открыть в IDE, прописать токен бота и запустить основной `main`-класс, если требуется посмотреть текущую реализацию.

## Статус проекта

Незавершённый эксперимент. Используется как «песочница» для идей.
